What do you use?
-React (Front)
-Bootstrap (Front)
-Express (Back)
-NodeJs (Back)
-Fontawesome (Front)
*Database: MySql (the script is in the database folder):
…\fullstack_node_react-main\node\database
Structure: 2 folders:
1) Para node y express(the server)
2) Para el front(interfaz de usuario)
*ORM(Sequelize) for MySql
What did I install?
-NodeJs -> https://nodejs.org/es/download/
-React -> https://es.reactjs.org/docs/getting-started.html
-Nodemon -> https://www.npmjs.com/package/nodemon
To run the app you must run nodemon and the script of the database!
